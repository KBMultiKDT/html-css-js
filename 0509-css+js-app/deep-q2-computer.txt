1. 프로그램 로드 (Load Program):
프로그램이 실행되기 전에 HDD에 저장된 프로그램이 RAM으로 로드됩니다.
이때 HDD에서 RAM으로 데이터를 이동시키는 역할을 하는 것이 바로 CPU입니다.
CPU는 프로그램을 HDD에서 읽어와 RAM으로 전송합니다.

2. 프로그램 실행 (Program Execution):
프로그램이 RAM에 로드되면 CPU는 프로그램의 명령어를 하나씩 실행합니다.
RAM은 CPU에 필요한 데이터를 제공하고, CPU는 RAM에서 데이터를 읽어와 연산을 수행하고 결과를 RAM에 다시 쓰기도 합니다.

3. 데이터 처리 (Data Processing):
CPU는 RAM에 있는 데이터를 읽어와서 계산하고, 필요한 경우에는 RAM에 쓰거나 HDD에 저장합니다.
RAM은 이러한 데이터 처리 과정에서 중요한 역할을 합니다.
CPU는 빠른 속도로 RAM에서 데이터를 읽고 쓸 수 있기 때문에 RAM에 데이터를 유지하고 처리하는 것이 효율적입니다.

4. 결과 저장 (Result Storage):
CPU가 데이터를 처리한 결과는 RAM에 일시적으로 저장될 수도 있고, HDD에 영구적으로 저장될 수도 있습니다.
일시적인 결과는 RAM에 저장되며, 프로그램이 종료되면 RAM에서 사라집니다.
영구적인 결과는 HDD에 저장되어 나중에 필요할 때 다시 사용할 수 있습니다.

5. 프로그램 종료 (Program Termination):
프로그램 실행이 완료되면 CPU는 해당 프로그램의 실행을 종료합니다.
RAM에 로드된 프로그램과 데이터는 삭제됩니다.

이처럼 CPU는 프로그램의 명령어를 실행하고,
RAM은 CPU에 필요한 데이터를 제공하고 중간 결과를 저장하며,
HDD는 프로그램과 데이터를 영구적으로 저장하는 역할을 합니다.
